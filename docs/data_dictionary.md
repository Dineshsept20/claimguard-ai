## 1. Claims (`claims.csv`)

The primary dataset. Each row is one pharmacy claim (prescription fill).

| Column | Type | Description | Example |
|--------|------|-------------|---------|
| `claim_id` | string | Unique claim identifier | `CLM00000001` |
| `service_date` | datetime | Date the prescription was dispensed | `2025-03-15` |
| `submit_hour` | int | Hour of day the claim was submitted (0–23) | `14` |
| `member_id` | string | FK → members.member_id | `MEM00001234` |
| `prescriber_id` | string | FK → prescribers.prescriber_id | `PRE0001` |
| `pharmacy_id` | string | FK → pharmacies.pharmacy_id | `PHR0001` |
| `ndc` | string | National Drug Code (11-digit) | `00093015305` |
| `drug_name` | string | Brand or trade name of the dispensed drug | `Lipitor` |
| `generic_name` | string | Generic / INN name | `Atorvastatin` |
| `therapeutic_class` | string | Drug therapeutic category (27 classes) | `STATIN` |
| `dea_schedule` | string | DEA schedule (`NONE`, `II`, `III`, `IV`, `V`) | `II` |
| `quantity` | int | Number of units dispensed | `30` |
| `days_supply` | int | Days the prescription should last | `30` |
| `refill_number` | int | Which refill this is (0 = new, 1+ = refill) | `3` |
| `ingredient_cost` | float | Cost of the drug ingredient ($) | `45.67` |
| `dispensing_fee` | float | Pharmacy dispensing fee ($) | `3.50` |
| `copay` | float | Member copayment ($) | `10.00` |
| `plan_paid` | float | Amount the plan/insurer paid ($) | `39.17` |
| `total_cost` | float | `ingredient_cost + dispensing_fee` ($) | `49.17` |
| `diagnosis_code` | string | ICD-10-CM diagnosis code | `E78.5` |
| `diagnosis_desc` | string | Human-readable diagnosis | `Hyperlipidemia, unspecified` |
| `prior_auth_flag` | bool | Whether prior authorization was required | `False` |
| `claim_status` | string | Adjudication result (`paid`, `rejected`, `reversed`) | `paid` |
| `is_brand` | bool | Whether the dispensed drug is a brand product | `False` |
| `mme_factor` | float | Morphine milligram equivalent factor (0 for non-opioids) | `1.5` |
| `is_anomaly` | bool | **Label:** whether this claim is anomalous | `False` |
| `anomaly_type` | string | **Label:** type of anomaly (or `none`) | `none` |

### Anomaly Type Values

| Value | Description |
|-------|-------------|
| `none` | Normal claim |
| `quantity_manipulation` | Inflated quantity (3–5× typical) |
| `prescriber_pharmacy_collusion` | Concentrated prescriber→pharmacy pipeline |
| `doctor_shopping` | Member visiting multiple prescribers for controlled substances |
| `therapeutic_duplication` | Same drug class from multiple prescribers |
| `phantom_billing` | Drug-diagnosis mismatch / billing for unrelated condition |
| `upcoding` | Brand dispensed instead of generic / price inflation |
| `refill_too_soon` | Refill before supply should be exhausted |
